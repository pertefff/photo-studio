Проект представляет из себя клиент-серверное приложение

Клиентское приложение (папка client) отвечает за отображение сайта в браузере для клиентов и отправляет запросы на сервер (папка server)

Административное приложение (папка admin) позволяет владельцам менять данные на сайте: цены, доступные залы для съёмок и тд. Написан на react-admin и общается с сервером
